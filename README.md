# Bookstore Inventory

Monorepo for a local-first bookstore inventory management system.

| Package | Description |
|---|---|
| `server/` | Hono + Drizzle + PostgreSQL backend |
| `client/` | React + Vite web dashboard |
| `mobile/` | Expo mobile app for barcode scanning |

## Getting started

```bash
# Start the backend
cd server && npm install && npm run dev
```
